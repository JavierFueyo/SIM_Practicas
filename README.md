Simulador de mortero: tu objetivo es golpear a los enemigos y tirarlos de la plataforma.

Controles:

- I -> Gira el mortero hacia arriba.
- K -> Gira el mortero hacia abajo.
- J -> Mueve el mortero hacia la izquierda.
- L -> Mueve el mortero hacia la derecha.
- P -> El mortero dispara.
- 0 -> Seleccionas “Bala Piedra”
- 1 -> Selecciona “Bala Dinamita”
- V -> Activa/Desactiva el viento
